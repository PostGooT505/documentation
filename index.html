How to upload your Publication Logos
🟡 Regular Publication Logo (Light Mode)
  1. Go to Pressrex > Dashboard from the admin menu.
  2. Click Customize your design in the same interface.
  3. Go to Settings > Design & Branding.
  4. Click Brand in the Design settings sidebar on the right.
  5. Click Upload logo in the Publication logo menu item.
  6. Click Site-wide in the Design settings sidebar on the right.
  7. Click Upload image in the Resize logo menu item.
⚫ Dark Mode Logo
  1. Go to Settings > Design & Branding from the admin menu.
  2. Click Site-wide in the Design settings sidebar on the right.
  3. Click Upload image in the Dark mode logo menu item.
  4. Click Upload image again in the Resize dark mode logo menu item.

🔥 Trends

Configure the appearance and behavior of the Trends section shown on your homepage.

Steps to Customize Trends
  1. Click on Theme in the Design menu (on the right).
  2. Use the "Trend Title" input to change the trend section title.
  3. Use the "Trend Tag" input to define the tag you want to show as trends.
  4. In the "Home Page Style" select box, choose any style for how trends will appear.

Tags

Principle includes a template for the Categories (Tags) page, which displays all the categories or tags of your publication. This page is optional, but you can add it if desired.

🛠️ How to create the Categories page:
  1. Go to Pages from the admin menu.
  2. Click the New page button in the top right corner.
  3. Enter a Page Title (e.g., Categories or Tags).
  4. Optionally, upload a featured image for this page.
  5. Click the ⚙️ icon in the top right corner to open Page Settings.
  6. Add a short description in the Excerpt field (optional).
  7. In the Template dropdown, select Categories.
  8. Click Publish to create the page.

Feature this post

  • 🖱️ Accédez au menu Posts dans la barre latérale gauche.
  • ➕ Cliquez sur New Post en haut à droite pour ouvrir l'éditeur de création de publication.
  • 🖼️ Repérez l'option radio "Feature this post".
  • 📌 Cette option permet de mettre en avant la publication, en la plaçant en première position sur la page d’accueil.

Code Injection

  • 🖱️ Accédez au menu Posts dans la barre latérale gauche.
  • ➕ Cliquez sur New Post en haut à droite pour ouvrir l'éditeur de création de publication.
  • 🖼️ Repérez l'option radio "Code Injection".
  • 📌 Cette option permet aux utilisateurs avancés d’injecter du code personnalisé (HTML, JavaScript, etc.) dans le thème. Cela peut être utile pour ajouter des fonctionnalités, mais présente un risque de sécurité si elle est mal utilisée.

Menu Navigation

  • 📂 Go to the Dashboard.
  • ⚙️ In the sidebar, click on Settings.
  • 🌐 Under the Site section, click on Navigation.
  • 🛠️ Click on Customize to edit the menu.
  • ➕ You can add a new item to the main menu.
  • 📌 Note: To add a submenu, add a dash (-) before the item.
  • 📎 To add another menu item, use a plus sign (+) before the submenu of the previous item.
